まずは公式サイト 👉 Google Antigravity 公式サイトはこちら にアクセスしてな。

  • 自分のPCに合ったものを選ぼう:

2. 初心者が迷わない「4つの初期設定」

起動した後の英語画面、これを選べば間違いなしや！

① Choose setup flow
  • Start fresh: まっさらな状態で新しくセットアップを始める（初心者へのおすすめ度 100 億点満点や！）
  • Import from VS Code: VS Code の設定や拡張機能をガツンとお引越しする
  • Import from Cursor: Cursor の設定や拡張機能をはんなりと引き継ぐ
  • Use VS Code extensions: VS Code の拡張機能だけを個別に選んで使う

ここは、迷わず 『Start fresh』 を選んでな。まっさらな状態で始めるのが一番スムーズや。

② Theme

好きなんでええけど、おすすめは 『Tokyo Night』。未来的でテンション上がるで！

③ Agent設定

左側選択肢の説明

  • Agent-driven development（AI主導開発）: AIが主体となってコードを書き進めるモード。自律的にガンガン動くプロ向けや！
  • Agent-assisted development（AIアシスト開発）: AIと相談しながら進めるバランス型。おすすめやで。
  • Review-driven development（レビュー主導開発）: AIが勝手に動く前に必ず確認を求めてくる。安心感が100億点満点や！

右側設定の説明

  • Terminal execution policy（ターミナル実行ポリシー）: AIがコマンドを実行する時のルールや。
    • Auto: AIが判断して自動でコマンドを実行する。作業がサクサク進むで。
    • Always require confirmation: コマンドを叩く前に毎回「ええか？」って聞いてくれる。慎重派の君におすすめや。
  • Review policy（レビューポリシー）: AIの指示を人間がチェックする頻度や。
    • Agent Decides: AIが「これは確認が必要や」と判断した時だけ聞いてくる。
    • Always Review: どんな小さな動きでも、必ず君の許可を求めてくるで。
  • JavaScript execution policy（JavaScrit 実行ポリシー）: ブラウザ操作などで使うJavaScriptをどう扱うかや。
    • Auto: 必要に応じてAIが自動で実行する。
    • Always require confirmation: 実行前に必ず確認のボタンが出るようになるわ。
  • Use the default allowlist for the browser: 信頼できる有名サイトへのアクセスを最初から許可するかどうかの設定や。チェックを入れておくと、ブラウザを使った調べ物がスムーズになるで。

ここはわからなければ 『Review-driven development』 を選択してな。AI が勝手に動く前に確認してくれるように促してくれるから、安心感が 100 億点満点や。右側もわからんかったらデフォルトのままでOKやで！

④ エディタの設定:Configure Your Editor

次はエディタの使い勝手を決める設定画面や！ ここもサクッと進めていこか。

keybindings（キーバインド）:
こだわりがなければ 『Normal』を選んどけば間違いないで！ もし君が熟練のVim使いなら『Vim』を選んでな。

Extensions（拡張機能）:
右側の 『Install 7 Extensions』にはぜひチェックを入れておこか。Agentの機能をフル活用するために必要な言語拡張機能が入るから、これを入れておけば安心感が100億点満点やで！

Command Line（コマンドライン）:
ここも 『Install』にチェックを入れておくのがおすすめや。これを設定しておくと、黒い画面（ターミナル）で `agy` って打ち込むだけで、いつでもAntigravityがパッと開けるようになるんや。めっちゃ便利やで！

全部チェックできたら、右下の 『Next』 ボタン（またはEnterキー）を押して次へ進むんやで〜！

3. Googleアカウントでログイン

最後に、Google アカウントでログインして準備完了や！

実は今、Antigravityは Google Workspace アカウントには対応してないみたい。無理やり繋ごうとすると大事なアカウントに「お仕置き（BAN）」が下るっていう物騒な噂も飛び交ってるんでホンマに気をつけてな。
