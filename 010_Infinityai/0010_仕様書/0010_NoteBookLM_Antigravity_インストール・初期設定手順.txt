# 【完全図解】Google Antigravityの導入ガイド！AIと創る未来の開発環境を手に入れよう

想像してみてください。あなたが頭の中で思い描いたアイデアを、言葉にするだけでAIが瞬時にアプリやWebサイトに変えてくれる未来を。

2025年11月、Googleが発表したAI搭載の次世代エディター「**Google Antigravity（アンティグラビティ）**」は、まさにその未来を現実にする魔法のツールです。プログラミングの深い知識がなくても、自律型AIエージェントがあなたの「相棒」となり、コードの記述からブラウザでの動作確認までを全自動でこなしてくれます。

この記事では、AIと協働する新次元のクリエイティブ体験を今すぐ始めるために、Antigravityのインストールから初期設定までを画像付き（※適宜画像を挿入してください）で徹底解説します！

特に初心者が英語の画面でフリーズしやすい**『Import Settings』『Theme』『Agent Mode』の3つの初期設定**については、「**結局どっちを選べば正解なの？**」という疑問に論理的かつ明確にお答えします。

さあ、あなたのアイデアが重力から解き放たれる瞬間を一緒に体験しましょう！

## ステップ1：公式サイトからダウンロード＆インストール

まずは、あなたのPCにAntigravityをインストールしましょう。

### 1. 自分のPCのスペックを確認する
公式サイト（https://antigravity.google/download ）にアクセスすると、ダウンロードボタンが表示されます。お使いのOSに合わせてインストーラーを選びましょう。

*   **🍎 Macユーザーの場合**:
    画面左上の「リンゴマーク」＞「このMacについて」を開き、チップを確認します。
    *   「Apple M1〜M4」などと書かれている → **「Apple Silicon」**を選択
    *   「Intel Core〜」と書かれている → **「Intel」**を選択
*   **🪟 Windowsユーザーの場合**:
    ほとんどのPCは**「x64」**でOKです。迷ったらx64をダウンロードしましょう。

### 2. インストールを実行する
*   **Mac**: ダウンロードした`.dmg`ファイルを開き、Antigravityのアイコンを「Applications」フォルダにドラッグ＆ドロップするだけです。
*   **Windows**: ダウンロードした`.exe`ファイルを実行し、画面の指示に従って「次へ」を押していけば完了です。

> 💡 **【重要】Windowsユーザーの方へ**
> 「Gemini 3.1 Pro is not available...」というエラーが出た場合は、**絶対にアンインストールせず、そのまま公式サイトから最新版を「上書きインストール」**してください。データや設定を消さずに、3分で最新モデルが使えるようになります。

---

## ステップ2：初心者が迷う「3つの初期設定」の正解はコレ！

インストールが終わってアプリを起動すると、英語の初期設定（セットアップ）画面が表示されます。「うわっ、英語だ…」と焦らなくても大丈夫！青い「Next」ボタンを押して進めながら、以下の3つの重要ポイントを押さえていきましょう。

### ① Import Settings (Choose setup flow)：どっちが正解？
最初の画面では、環境をどう構築するかを聞かれます。

*   **選択肢**: 「Start fresh (新規開始)」 または 「Import from VS Code / Cursor (設定の引き継ぎ)」
*   **🎯 正解**: 初心者は迷わず **『Start fresh』** を選択！

**【論理的な理由】**
VS CodeやCursorといった他のエディターをバリバリ使いこなしているプログラマーであれば、設定を引き継ぐ「Import」が便利です。しかし、AIエディターを初めて触る方や、まっさらな状態で心機一転始めたい方は、不要な設定が混ざらない「Start fresh」を選ぶのが最もトラブルが少なく、スムーズに学習を始められるからです。

### ② Theme (Choose an editor theme type)：どっちが正解？
次に、作業画面の「見た目（色合い）」を選びます。

*   **選択肢**: 「Dark」「Tokyo Night」「Light」など
*   **🎯 正解**: 直感で選んでOKですが、おすすめは **『Tokyo Night』** または **『Dark』**！

**【論理的な理由】**
完全に好みの問題なので後からでも変更可能ですが、プログラミングの世界では「背景が暗い画面（ダークテーマ）」が圧倒的なスタンダードです。長時間画面を見つめても目が疲れにくく、コードの色分け（シンタックスハイライト）が美しく際立つからです。特に「Tokyo Night」は背景が真っ黒でパステルカラーの文字が映えるため、未来的でテンションが上がりますよ！

### ③ Agent Mode (How do you want to use the Antigravity Agent?)：どっちが正解？
**ここが最も重要な設定です！** AIに「どこまで自由に行動させるか（権限）」を決めます。

*   **選択肢**: 
    1. Agent-driven development (AIにすべてお任せ)
    2. Review-driven development (AIが提案し、人間が承認する)
    3. Strict Mode / Secure Mode (超厳格モード)
    4. Custom configuration (カスタム設定)
*   **🎯 正解**: 初心者は必ず **『Review-driven development (推奨)』** を選択！

**【論理的な理由】**
AntigravityのAIは非常に優秀で、放っておいてもファイルを作成したりターミナルでコマンドを実行したりします。しかし、AIに全権限を渡す「Agent-driven」を選ぶと、AIが暴走して大切なファイルを勝手に消してしまったり、意図しないコードを実行してしまうリスクがあります。
「Review-driven」を選べば、**AIが何か重要な操作をする前に必ず「実行してもいいですか？」とあなたに確認（Review）を求めてきます**。安全性とスピードのバランスが最も良く、AIが何をしているのかを学びながら開発を進められるため、初心者にとっての最適解なのです。

## ステップ3：その他の設定〜Googleアカウント連携

3つの難関を越えれば、あとは簡単です！

1.  **Configure Your Editor**: キーボード設定（Keybindings）などは「Normal」、拡張機能は「Recommended」のまま、デフォルト設定で「Next」を押します。
2.  **Sign into Google**: 「Sign in with Google」をクリックし、普段お使いのGoogleアカウントでログインします。
3.  **Terms of Use (利用規約)**: 内容を確認してチェックを入れ、「Next」を押せば初期セットアップは完了です！ （※機密情報などは入力しないよう注意しましょう）

## 仕上げの魔法：英語画面を「日本語化」しよう！

起動した画面はまだ英語ですが、たった1分の魔法で日本語の快適な画面に変えることができます。

1.  画面左端のメニューバーから、四角いブロックのアイコン（Extensions / 拡張機能）をクリックします。
2.  検索バーに「**Japanese**」と入力します。
3.  地球儀のアイコンの「**Japanese Language Pack for Visual Studio Code**」を選び、「Install」ボタンを押します。
4.  インストール完了後、画面右下に表示される「Change Language and Restart（言語を変えて再起動）」をクリックします。

アプリが再起動すると……見慣れた安心の「日本語」メニューに生まれ変わっています！ これで、英語アレルギーの方でも安心してAI開発に没頭できますね。
