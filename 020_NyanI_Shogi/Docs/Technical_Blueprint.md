# にゃん囲将棋 (Nyan-i Shogi) - Technical Blueprint

## 🏗️ システム構成
「にゃん囲将棋」は、ブラウザだけでサクサク動く、最高にモダンなWebアプリとして構築します。

### ステップ1: 癒やしのプロトタイプ (V1.0)
- **基盤**: HTML5 + Vanilla CSS (Glassmorphism) + JavaScript
- **目的**: まずは「見た目の可愛さ」と「触り心地の良さ」をボスに体感してもらう。
- **特徴**: 
    - 京都の静寂を感じる「禅ガーデン」背景。
    - 駒を動かすたびに「肉球」のエフェクトが出る。
    - にゃんこ（エージェント）が画面の端で応援してくれる。

### ステップ2: インテリジェント・エンジン (V2.0)
- **AI連携**: WordPress経由のMCP（Project Kyotoの脳）と接続。
- **機能**: 
    - 「にゃんこ囲い」の自動認識。
    - Un-chanによる対局分析。

## 🎨 デザインガイドライン
- **Color Palette**: 
    - Sakura Pink (#FFD1DC) - 成功エフェクト
    - Matcha Green (#8BA989) - 盤面ハイライト
    - Dark Kyoto (#1A1A1A) - 夜間モード（高級感）
- **UI Element**: 全てのパネルは「半透明の磨りガラス（Glassmorphism）」で統一。
