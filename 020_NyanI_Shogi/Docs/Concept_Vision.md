# にゃん囲将棋 (Nyan-i Shogi) - Project Vision

## 概要
「にゃん囲将棋」は、初心者から上級者までが楽しめる、世界で最も「可愛くて」「癒される」将棋アプリケーションです。
プロジェクト・キョウトの理念「テクノロジーの力で苦労を楽にする」に基づき、将棋の難解なイメージを払拭し、猫たちが可愛く対局をサポートします。

## ターゲット
- 将棋に興味はあるが、ルールや囲いが難しそうと感じている人。
- 可愛いキャラクター（猫）に癒されながら対局したい人。
- 新しいスタイルの将棋体験を求めているファン。

## 主要機能（フェーズ1）
- **にゃんこ囲いサポート**: 複雑な「囲い」を、可愛い猫たちが合体するアニメーションで視覚的にガイド。
- **癒やしの対局UI**: 京都の四季を感じる背景と、心地よい駒の音。
- **エージェント・アシスト**: MittanやUn-chanたちが、対局中の「次の一手」をユーモアを交えてヒント。

## プロジェクト構成
- `020_NyanI_Shogi/`
    - `Docs/`: 仕様書、概念図、マニュアル
    - `Assets/`: 猫キャラクター、駒のデザイン、背景画像
    - `Src/`: プログラムソース（Next.js + Tailwind CSS を想定）
